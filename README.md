This folder contains Week 4 theoretical learning and practical SOC activities completed using Ubuntu and Splunk.

- Alert-Triage
- Evidence-Analysis
- Post-Incident-Analysis
- Security-Metrics
- SOAR-Playbook

## Workflow Followed
1. Reviewed Ubuntu auth and system activity.
2. Performed threat hunting using network and process checks.
3. Triaged failed events from the authentication log.
4. Collected supporting evidence from the Ubuntu VM.
5. Added /var/log/auth.log into Splunk for monitoring.
6. Validated failed events in Splunk.
7. Performed post-incident review and metrics analysis.
8. Documented a SOAR playbook and adversary-style detection scenario.
9. Combined all activities into a capstone incident-response workflow.
